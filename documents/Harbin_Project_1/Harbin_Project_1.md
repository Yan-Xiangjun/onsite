1 工程概况
1.1 项目名称：哈尔滨市道外区南坎街11—4号房屋解危维修加固项目
1.2 工程地点： 哈尔滨市道外区南大街11-4号
1.3 项目概况：
1.3.1 既有结构体系为 砌体结构；主要竖向承重构件为 砖墙 ；主要水平承重构件为混凝土梁、现浇楼板和木屋架 ；基础形式 不详 。
1.3.2 鉴定报告类型为房屋危险性鉴定，房屋评定等级为D级。
1.3.3房屋建设年代1965年。
1.4房屋存在的问题及解危处理措施：
1.4.1依据《道外区南坎街11-4号房屋危险性鉴定报告》所述，现房屋存在以下主要问题。
1）山墙处存在多条因地基不均匀沉降所导致的斜向贯通裂缝，现地基变形沉降已基本趋于稳定。
2）木屋架普遍存在腐烂、槽朽现象；黏土平瓦屋面破损，已丧失防水功能。
以上问题均为鉴定报告提出的内容,但鉴于检测时房屋检测环境受限等原因,避免不了有遗漏的裂缝、变形等损伤。故施工时前,应对房屋进行全面排查,若存在以上问题但本图纸未提到的部位或有新的损伤和变形存在,应及时通知甲方和加固设计单位补充加固方案。
1.4.2针对本说明第1.4.1条房屋存在的问题采取以下加固方法或处理措施。
1)针对存在沉降裂缝墙体,采取先对墙体灌浆补强,再采取钢筋混凝土面层的加固措施。针对下部地基,采取注浆补强加固。
2）针对木梁、木板和木屋架腐烂、槽朽的构件，采取局部补强或置换的加固措施。针对黏土平瓦屋面破损的情况，采取更换屋面措施。

7植筋专项说明
1、施工顺序
钢筋下料 → 标定原构件钢筋位置 → 定位放线 → 钻孔 → 孔洞清理 → 钢筋除锈 → 注胶植筋 → 固化养护 → 质量检查与验收。
2、施工工艺
1）钢筋下料
根据结构施工图，进行钢筋下料，下料时需考虑钢筋的截断、搭接等问题，使之满足《混凝土结构工程施工质量验收规范》（GB50204-2015）的相关规定。
2）标定原构件钢筋位置
采用钢筋探测器等设备，扫描原结构构件中钢筋，并标定位置。
3）定位放线
依据结构施工图，确定植筋位置，植筋位置不仅要避让原构件中的钢筋，还要满足设计及规范的相应要求。经调整的植筋位置需提供给设计院核定计算和构造满足要求后再进行下一步施工。
4）钻孔
钻孔前应剔凿掉装饰层（包括抹灰层），露出结构构件基层，用冲击钻在原结构构件上钻孔，钻孔深度按设计要求确定。若构件有混凝土剔除（凿毛）厚度要求，以剔凿后混凝土面作为植筋深度计算起始位置。对于产生的废孔，应采用Ⅱ级水泥基灌浆料或Ⅰ级聚合物改性水泥砂浆灌实。
5）孔洞清理
先用毛刷清理孔壁，把灰尘或碎渣清理掉，再用吹风机等类似功能的工具吹出孔壁上的浮灰和细小颗粒，注胶前用棉丝沾丙酮再清理一次孔壁，以免影响结构胶与混凝土的有效粘结。如孔内有积水，应用棉丝将积水清理干净，然后采用自然风干或加热棒烘干等方法进行干燥处理。
6）钢筋除锈
当钢筋表面有锈蚀时，使用角磨机配电动钢丝刷除锈或采取其它除锈方法。钢筋锚固段用丙酮或无水乙醇擦拭干净后方可使用。
7）注胶植筋
锚孔深度大于 200mm 时，可采用混合管延长器注胶；注胶应从孔底向外均匀、缓慢地进行，应注意排除孔内的空气，注胶量应以植入钢筋后略有胶液被挤出为宜。而后将钢筋缓慢地按单一方向边转边插，直至达到规定的深度。从注入胶粘剂至植好钢筋所需的时间，应少于产品使用说明书规定的适用期（可操作时间），否则应拔掉钢筋，并立即清除失效的胶粘剂，按原工序重新安装。植入的钢筋应立即校正方向，并应保证植入的钢筋处于孔洞的中心位置。溢出孔外的胶体应及时进行清理。
8）固化养护
依据所采用的结构胶，按照产品说明书的要求进行固化，在固化的时间内保护钢筋不得受搅动，待结构胶固化后方可受力。
3、技术要求及注意事项
1）植筋应进行锚固承载力现场检验，检验应选取适当位置采用随机抽样办法取样；同规格、同品种、同强度等级的锚固件安装于锚固部位基本相同的同类构件为一检验批，抽取数量按每批锚固件总数的0.1%且不少于3件进行检验。
2）植筋非破坏性检验时应依据《混凝土结构后锚固技术规程》JGJ145-2013的要求执行，荷载检验值应取 0.9Afrk，试验时应采用分级加载，应将设定的检验荷载均分为10级。
3) 植筋施工时, 基材温度和孔内表层含水率应符合设计和胶粘剂使用说明书要求, 无明确要求时, 基材表面温度不应低于 15° C; 植筋施工严禁在大风、雨雪天气露天进行。
4）植筋钢筋宜采用机械连接接头，也可采用焊接连接，连接接头的性能应符合国家现行相关标准的规定。采用焊接接头时，焊接宜在注胶前进行，确需后焊接时，应进行同条件焊接后现场破坏性检验；焊接施工时，应断续施焊，施焊部位距离注胶孔顶面的距离不应小于20d，且不应小于200mm，同时应用水浸渍多层湿巾包裹植筋外露部分，钢筋根部的温度不应超过胶粘剂产品说明书规定的最高短期温度；焊接时，不应将焊接的接地线连接到植筋的根部。
5）本说明未提及的内容应按《建筑结构加固工程施工质量验收规范》（GB50550-2010）要求执行。

8 钢筋混凝土面层加固法专项说明
# Block 42
1.加固范围详见加固平面图。
2. 设计构造
1）加固面层材料为C30补偿收缩细石混凝土，加固面层厚度为70mm。
2）采用钢筋混凝土面层加固墙体时，钢筋网外保护层厚度不应小于 15mm ，钢筋网片与墙面的空隙不应小于 10mm 。
3. 施工要点
1）面层加固法的施工工艺流程为：原有墙面清底一钻孔并用水冲刷一孔内干燥后安设锚筋并铺设钢筋网一浇水湿润墙面一喷射或浇筑混凝土并养护一墙面装饰。
2）原墙面碱蚀严重时，应先清除松散部分并用M10或1：3水泥砂浆抹面，已松动的勾缝砂浆应剔除。
3) 在墙面钻孔时, 应按设计要求先画线标出锚筋或穿墙筋的位置, 并应采用电钻在砖缝处打孔, 穿墙孔直径宜比S形筋大2mm; 锚筋孔直径宜采用锚筋直径的1.5~25倍, 其孔深大于等于120mm,
4）铺设钢筋网时，竖向钢筋应靠墙面并采用钢筋头支起钢筋网片与墙面的空隙宜大于等于 10mm，钢筋网外保护层厚度大于等于 15mm。

加固设计说明：
1.木屋架修复方法：
(1) 对于构件细小轻微的裂缝（在 0.5~cm 以内，包括天然小裂缝），可用环氧树脂腻子堵抹严实。
(2) 裂缝宽度（超过 0.5~cm，在 3~cm 范围以内）可用木条粘牢补严，直接用铁箍加固，铁箍的大小与数量，可根据具体情况确定，铁箍接头处用螺栓或特制大帽钉联结牢固使裂缝闭合。
(3) 对于裂缝较宽在 3~cm 以上（应在构件直径的 1 / 4 以内）深达构件截面中心的粘补木条后，还要根据裂缝的长度加铁箍 2~3 道。嵌补的木条最好用顺纹通长，断面较大的矩形构件可用U形铁兜绊，上部用长脚螺栓拧牢。铁箍可选厚度 3~10~mm，宽度 50~150~mm，间隔一定距离设置铆钉。
(4) 对于超出上述裂缝范围或腐朽严重的木构件应按原截面规格及材料进行更换处理。所有外露钢材涂刷红丹防锈漆两道。
（5）对破损或锈蚀屋面板应进行更换处理。
